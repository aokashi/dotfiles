# Windows 環境構築手順

1. Scoop をインストール (`scoop_install.ps1`)
2. Windowsの設定から機能追加の画面に移り、 OpenSSH クライアントをインストール
3. Microsoft Store から必要なソフトウェアを取得、インストール
4. nvm の設定を済ませ、 yarn を scoop でインストール
    - https://github.com/coreybutler/nvm-windows/
